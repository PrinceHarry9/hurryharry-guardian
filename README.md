# HurryHarry Guardian

AI-powered safety and monitoring system.

## What is HurryHarry Guardian?

HurryHarry Guardian is an AI-based mobile safety application designed to detect potential threats in real time using computer vision and smart detection technology.

The system uses:

- AI object detection
- Real-time camera monitoring
- Safety alerts
- Cloud processing
- Mobile integration

The goal is to improve personal and public safety by detecting suspicious situations faster than human response.

## Technology Stack

- Python
- YOLO object detection
- Flask API backend
- Android mobile app
- Computer vision
